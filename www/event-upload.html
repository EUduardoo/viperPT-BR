Import an Event

  1. Go to The Blue Alliance website
  2. Navigate to the event page on that site for the event you want to import.
  3. Use Ctrl-u to view the web page source for that event page.
  4. Use Ctrl-a to select the entire source code.
  5. Use Ctrl-c to copy all the selected source code.
  6. Use Ctrl-v to paste it all into the text area below.
  7. Click the 'Upload Event' button.
